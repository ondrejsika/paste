Documentation
-------------

### Installation

```
git clone git@github.com:ondrejsika/time-keeper.git
cd time-keeper
virtualenv env
./env/bin/pip install -r requirements
./manage.py syncdb
./manage.py migrate
```

### Run

```
./manage.py runserver
```
